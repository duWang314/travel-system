TravelOS
推荐 地图路线 游记交流 室内导航 用户中心

数据结构课程设计 · 无数据库版

把 200 个目的地、道路图与游记压缩算法装进一个旅游系统。

系统覆盖旅游推荐、地图路线规划、附近设施查询、室内导航、游记全文搜索、哈夫曼压缩、美食推荐和管理员管理，所有数据均以 JSON/TXT 文件持久化。

开始推荐 查看算法说明

Heap Top-K Recommendation

个性化旅游推荐

通过热度、评分和兴趣匹配构成加权得分，再使用小顶堆选出 Top 10，不对全部 200 条目的地记录完全排序。

Dijkstra Graph Navigation

景区 / 校园地图与路线规划

SVG 会直接渲染 JSON 道路图；起点、终点、必打卡点、交通工具与路径目标可动态调整。附近设施按道路最短路径距离排序，不采用直线距离。

请选择目的地并规划路线。

Food Ranking

附近美食推荐

美食模块支持菜系、关键词和目的地过滤，使用评分、热度与距离构成综合得分并返回 Top 10。

KMP Search · Huffman Compression

游记管理与交流

标题精确查找使用哈希索引，正文全文检索使用 KMP，游记压缩使用字符级哈夫曼编码，支持评分、浏览量和封面提示词生成。

Indoor Navigation

公共教学楼室内导航

从景区/校园道路图中可进入公共教学楼。这里使用室内节点图继续运行 Dijkstra，生成入口到目标房间的分步指引。
